## Featured Work

#### [NYC Traffic Collison Prediction](https://github.com/omkar-waikar/NYC-Traffic-Collison-Prediction.git) | **Real time Distributed Data Pipeline & Predictive Analytics Platform**
* **Scalable Data Ingestion:** Designed a high velocity **Apache Kafka** producer-consumer pipeline to ingest live NYC Open Data API streams and implementing custom watermarking and retry logic for zero loss data integrity.
* **Distributed ML Processing:** Leveraged **Dask** for distributed data handling and **Scikit-learn** to build a predictive model mapping collision likelihood. Utilized **Ball Trees** for high performance geo-spatial coordinate mapping across massive datasets.
* **Optimized Architecture:** Managed stateful data storage with **MongoDB** and utilizing geographical indexing to achieve sub second query latency for live traffic speed lookups.

#### [Deepfake Detection System](https://github.com/omkar-waikar/Deepface-Deepfake-Detection-System.git) | **AI Security Platform for Multimedia Deepfake Detection**
* **Hybrid Deep Learning Engine:** Architected a detection suite leveraging both **PyTorch and TensorFlow** to perform real time analysis of image, audio, and video streams with high confidence scoring metric.
* **Cloud Deployment:** Engineered and deployed the full stack application on **AWS** ensuring high availability for compute intensive computer vision and processing tasks.
* **Secure API Architecture:** Designed RESTful APIs with **JWT based authentication** implementing secure user workflows to maintain data privacy and individual user sessions.
